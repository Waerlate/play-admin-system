# play-admin-system

基于 Vue3 + Element Plus + Node.js + Express + MySQL 构建的企业级后台管理系统。

## 技术栈
- Vue3
- Element Plus
- Pinia
- Vue Router
- Node.js
- Express
- MySQL
- JWT
- Joi

## 已实现功能
- 用户注册/登录
- JWT 身份认证
- 基于角色的权限控制（admin/user）
- 用户管理 CRUD
- 分页与搜索
- 全局错误处理中间件
- 参数校验
- GitHub 项目管理
